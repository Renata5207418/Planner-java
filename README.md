# Planner de Tarefas em Java

Projeto simples desenvolvido em Java como parte do meu aprendizado da linguagem.

O objetivo é criar um planner de tarefas executado pelo terminal, praticando conceitos como:

- Classes
- Objetos
- Métodos
- Encapsulamento
- ArrayList
- Scanner
- Estruturas condicionais
- Laços de repetição

## Funcionalidades atuais

- Adicionar uma nova tarefa
- Listar tarefas cadastradas
- Encerrar o programa

## Estrutura do projeto

```txt
planner-java/
├── Main.java
├── Tarefa.java
├── README.md
└── .gitignore
````

## Como executar

Compile os arquivos:

```bash
javac Main.java Tarefa.java
```

Execute o programa:

```bash
java Main
```

## Exemplo de uso

```txt
=== Planner de Tarefas ===
1. Adicionar Tarefa
2. Listar Tarefas
0. Sair
Escolha uma opção:
```
